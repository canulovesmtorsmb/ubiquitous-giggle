ЕГЭ по литературе 2026 — интерактивный учебник-тренажёр
=========================================================

КАК ОТКРЫТЬ
-----------
Откройте index.html двойным щелчком в любом браузере (Chrome, Edge, Yandex, Firefox).
Это полностью рабочий сайт в одном файле: на главной — все произведения кодификатора
ФИПИ, клик по плитке открывает курс в той же вкладке, кнопка «← Все произведения»
возвращает назад. Интернет нужен только для показа иллюстраций (с Викисклада);
текст, задания и навигация работают офлайн.

ЧТО ВНУТРИ
----------
index.html  — весь сайт (64 курса: разборы, фрагменты, тесты, сочинения,
              банк сочинений и сопоставлений, цитатник, пробный вариант).
images/     — иллюстрации к курсу «Война и мир».

КАК ОПУБЛИКОВАТЬ В ИНТЕРНЕТЕ (GitHub Pages)
-------------------------------------------
1. github.com → New repository → Public.
2. Add file → Upload files → перетащите index.html и папку images → Commit.
3. Settings → Pages → Branch: main, папка / (root) → Save.
4. Через минуту появится ссылка https://ваш-логин.github.io/название/
